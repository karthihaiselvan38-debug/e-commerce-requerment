# E-Commerce Order Management Database System
# Project Overview

This project is a Database Design and Analysis System for an E-Commerce Order Management platform.
It focuses on understanding business requirements and designing a structured database system to manage customers, products, orders, payments, shipments, and reviews.

# Objective

The main objective of this project is to analyze the business requirements of an e-commerce system and prepare a Software Requirement Specification (SRS) document before database design begins.

# Business Scenario

A growing e-commerce company needs a centralized database system to manage:

Customer registration and profiles
Product and category management
Supplier details
Order processing
Payment handling
Shipment tracking
Customer reviews

The system improves efficiency, reduces data redundancy, and ensures accurate business reporting.

# Constraints
